# Auditor Wiki

This wiki contains generated operating guidance for Trainovate's ISO 14001 audit platform.

## Manuals

- [Auditor Implementation Manual](auditor-implementation-manual.md)
- [Auditor Training Manual](auditor-training-manual.md)

For the end-user, screen-by-screen **product** manual (Auditor, Auditee and
Administrator guides), see the [User Manual](../user-manual/README.md). The wiki
manuals above are firm-implementation and training guidance; the User Manual is
the hands-on "how to use each screen" reference and is updated as features
change.

## Content Boundaries

These manuals are Trainovate-authored guidance. They do not reproduce ISO standard requirements text. Use clause identifiers and short titles only unless Trainovate confirms licensing and legal approval.

## Intended Users

- Tenant Admins setting up an audit firm workspace.
- Lead Auditors planning and signing audit work.
- Auditors capturing evidence and drafting findings.
- Platform operators maintaining shared configuration.
