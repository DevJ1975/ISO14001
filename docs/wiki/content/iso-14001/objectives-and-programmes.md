## Good objectives

Environmental objectives should be consistent with the policy, **measurable** (where practicable), monitored, communicated and updated as needed. They should address significant aspects, compliance obligations, and risks and opportunities.

## Planning to achieve objectives

For each objective, document the action plan (cl. 6.2.2):

| Field | Example |
| --- | --- |
| Objective | Cut process water use 15% vs 2025 baseline |
| Target / metric | m³ per unit produced |
| Baseline | 2025 actuals |
| Actions | Install closed-loop rinse; fix leaks |
| Resources | Capital budget, maintenance hours |
| Responsible | Site engineer |
| Timeframe | By 2026-12-31 |
| How results are evaluated | Monthly meter reads vs target |

## Auditing objectives

The auditor checks that objectives exist for the right priorities, that plans are resourced and on track, and that progress is monitored with real data — not just stated intentions.
