## Operational control (cl. 8.1)

Establish controls to manage significant aspects and meet compliance obligations, applying a hierarchy:

1. **Eliminate / substitute** the hazard or aspect where possible.
2. **Engineering controls** — containment, abatement, automation.
3. **Administrative controls** — procedures, permits-to-work, competence.
4. **Monitoring** — verify the control is working.

Controls should also consider the **lifecycle perspective** — procurement requirements, design, use, transport and end-of-life.

### Outsourced processes

Where processes are outsourced or controlled by suppliers, define the type and extent of control and communicate relevant requirements.

## Emergency preparedness & response (cl. 8.2)

Identify credible emergency scenarios and prepare to respond:

| Field | Example |
| --- | --- |
| Scenario | Bund failure / chemical spill |
| Prevention | Tank inspections, secondary containment |
| Response procedure | Isolate, contain, notify, clean up |
| Roles | Spill response team, shift supervisor |
| Resources | Spill kits, PPE, drainage plans |
| Last drill / test | Date and outcome |
| Lessons learned | Actions arising |

Test the response periodically, learn from drills and actual incidents, and update the procedures.
